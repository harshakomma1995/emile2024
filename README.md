# Cloud Service

## Dependencies

- **Node** - `^12.*.*`
- **NPM** - `^6.*.*`
- **Postgres** - `^11.*`
- **Redis** - `^6.*`

## Getting Started

1. Createe database locally with name - `extramile`
2. Install dependencies
   ```
   npm install
   ```
3. Deploy database
   ```
   npm run db:deploy
   ```
4. Migrate database
   ```
   npm run db:migrate
   ```
5. Start the server
   ```
   npm run dev
   ```
